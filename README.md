## Project Description

Interactive Periodic Table & Molecule Builder is a React.js application that allows users to select chemical elements, build simple molecules, view molecular formulas, identify known compounds, calculate molecular weights, and save created molecules.

The project demonstrates React component-based architecture, state management, dynamic rendering, event handling, and user interaction.

---

## Features

* Interactive Periodic Table
* Molecule Builder
* Dynamic Formula Generation
* Known Molecule Detection
* Molecular Weight Calculation
* Save Molecule Functionality
* Delete Saved Molecules
* Undo Last Atom
* Clear Molecule
* Responsive User Interface

## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/kaustubh7770/periodic-table-builder.git
```

Navigate into the project:

```bash
cd periodic-table-builder
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
